# Poetic-Surveillance

Un système d'analyse en temps réel combinant détection de landmarks, analyse d'émotions, et effets visuels dégradés progressivement.

# Description
Ce projet implémente un pipeline complet de vision par ordinateur pour l'analyse faciale avec une interface de traitement d'image qui évolue au fil du temps. Le système extrait des métriques faciales, applique des transformations visuelles et génère des descriptions textuelles basées sur les caractéristiques détectées.
